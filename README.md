# Python File Renamer

Simple Python script to automatically rename files inside a folder.

## Features

- Automatically renames files
- Keeps original file extensions
- Easy to use
- Beginner-friendly

## Example

Before:

photo.png
document.txt
music.mp3


After:
file_1.png
file_2.txt
file_3.mp3


## How to Use

1. Install Python
2. Edit the folder path inside the script
3. Run the script

## Technologies

- Python
- os module
